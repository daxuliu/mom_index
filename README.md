# 👩‍👧 宝妈指数 (Mom Index)

追踪中文社交平台上散户/小白/宝妈的投资讨论热度。**指数越高，散户情绪越极端，市场越危险**——基于"擦鞋童理论"的行为金融学反向指标。

> **当菜市场大妈和宝妈都在讨论股票，就是该离场的时候。**
> — Joseph Kennedy, 1929

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 📊 **11 板块散户情绪看板** | 纳指/黄金/CPO/半导体/中韩半/小米/腾讯/美团/恒科/红利/美光 |
| 🔴 **跨境 ETF 溢价监控** | 纳斯达克100/标普500/日经225 共 20 只 ETF 实时溢价排行 |
| 📈 **实时报价联动** | 11 板块 + 20 只 ETF 30 秒刷新（腾讯免费 API）|
| 🔍 **任意股票动态查询** | 输入代码自动识别 A 股/港股/美股，实时抓股吧+情绪分析 |
| 📸 **一键分享海报** | 板块/ETF 风险提示海报（Canvas 原生生成）|

| 💬 **代表性小白帖** | 每板块展示典型"小白发言"+AI 解读 |

## 🚀 快速开始

### 本地开发

```bash
cd ~/Desktop/mom-index

# 1. 安装依赖（仅 flask + requests）
pip install -r requirements.txt

# 2. 启动后端（端口 8766，含 /api/query 和 /api/etf-nav）
python3 api_server.py

# 3. 运行数据采集 + 分析 + 指数计算（每天跑一次）
python3 pipeline.py

# 4. 浏览器打开
#    默认入口：    http://localhost:8766/
#    散户看板：    http://localhost:8766/dashboard.html
#    跨境 ETF：   http://localhost:8766/etf_premium.html
#    个股查询：    http://localhost:8766/query.html
```

### 部署到 Vercel（推荐）

```bash
# 1. 推送到 GitHub
git init && git add . && git commit -m "init"
git remote add origin <你的repo>
git push -u origin master

# 2. 在 vercel.com 点 "New Project" → 选 repo → Deploy
# 3. 完成！访问 https://<project>.vercel.app
```

**路由配置**（`vercel.json`）：
- `/api/*` → Flask serverless
- `/dashboard.html` `/etf_premium.html` `/query.html` → 静态文件
- `/` → 自动跳转 dashboard

免费额度：每月 100GB 流量 + 100 万次 API 调用。

## 🧮 分析方法

### 小白判定（40+ 信号词库）

每条帖子匹配多维度信号，每条判定带可读推理：

```
帖子「黄金亏了20%了要不要割肉啊😭」
  命中: 决策依赖(+7) + 情绪恐慌(+5)
  意图: 🔴 卖出（命中"割肉""亏了"）
  得分: 58分 → 判定「纯小白」
```

信号维度：

| 维度 | 权重 | 示例 |
|------|------|------|
| 身份自述 | +8 | "小白"、"宝妈"、"新手" |
| 知识求助 | +6 | "怎么买"、"在哪看"、"请教" |
| 决策依赖 | +7 | "该不该"、"要不要"、"还能买吗" |
| 情绪恐慌 | +5 | "亏麻了"、"好慌"、"心态崩了" |
| 跟风行为 | +6 | "听博主说"、"朋友推荐" |
| 过度乐观 | +4 | "梭哈"、"满仓干"、"稳赚" |
| 专业术语 | -5 | PE/PB/估值/溢价率（扣分项）|

### 指数公式

```
宝妈指数 = 小白占比×0.40 + 小白强度×0.25 + 情绪极端度×0.20 + 信号纯度×0.15

宝妈买入 = 买入小白占比×50 + 小白热度×30 + 买入强度×20
宝妈卖出 = 卖出小白占比×50 + 小白热度×30 + 卖出强度×20
```

### 跨境 ETF 溢价

```
溢价率 = (场内实时价 - T-1 净值) / T-1 净值 × 100%
```

> 注：使用 T-1 净值（而非 IOPV 估算净值）才是业内标准，
> IOPV 会低估溢价 0.3-0.5%，导致错过危险信号。

## 🛡️ 反检测系统

`collectors/anti_detection.py` — 三件套：

| 能力 | 说明 |
|------|------|
| **请求头轮换** | 7 个 Chrome/Edge UA 随机切换 + Sec-CH-UA 指纹头 |
| **人类延迟** | 高斯分布，每 10 次停 5-15s，每 50 次停 30-60s |
| **隐身脚本** | 8 条 — webdriver/plugins/hardwareConcurrency 全部伪造 |
| **Playwright 参数** | 20+ 启动参数隐藏自动化痕迹 |

## ⚠️ 已知局限

- **股吧小白占比低**：5-20% 是正常的，真正的小白信号需要小红书数据
- **关键词规则限制**：非 LLM 语义理解，会漏掉隐含信号、误判部分 spam
- **缺少回测**：尚未用历史行情数据验证指数与市场顶底的相关性
- **腾讯 API 字段可能变**：免费接口不稳定，注释里有字段索引
- **节假日 ETF 数据为空**：腾讯节假日停推数据，页面会显示"暂无"
